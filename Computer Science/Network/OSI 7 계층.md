##### L3) 네트워크(Network)

> 라우터, IP

데이터를 목적지까지 가장 안전하고 빠르게 전달하는 기능을 담당한다.

라우터를 통해 이동할 경로를 선택하여 IP 주소를 지정하고, 해당 경로에 따라 패킷을 전달해준다.

##### L4) 전송(Transport)

> TCP, UDP

TCP와 UDP 프로토콜을 통해 통신을 활성화한다. 포트를 기반으로 분류하고 라우팅하는 역할을 한다.

- TCP : 연결 · 순서 · 전달 보장을 제공

- UDP : 비연결형 · 비보장형 통신을 제공

##### L5) 세션(Session)

> API, Socket

데이터가 통신하기 위한 논리적 연결을 담당한다.

두 시스템 간의 세션을 설정 · 유지 · 동기화 · 종료하며, 통신의 흐름과 상태를 관리하는 계층이다.

##### L6) 표현(Presentation)

> JPEG, MPEG 등

데이터 표현에 대한 독립성을 제공하고 암호화하는 역할을 담당한다.

파일 인코딩 / 디코딩, 데이터 구조화 / 역구조화, 압축 / 해제, 암호화 / 복호화를 수행한다.

##### L7) 응용(Application)

> HTTP, FTP, DNS 등

최종 목적지로, 응용 프로세스와 직접 관계하여 일반적인 응용 서비스를 수행한다.

사용자 인터페이스, 전자우편, 데이터베이스 관리 등의 서비스를 제공한다.

현대 네트워크에서는 L5 ~ L7이 응용 계층(L7)에 흡수된 경우가 많다.

> HTTP 세션, 로그인 세션, WebSocket 세션, JWT / Session ID 등
